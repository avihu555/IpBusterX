IP Buster app

This is a tool for cross info from 3 API sources: ABUSEIPBD, OTX, VirusTotal

How to use:

1: Please sign in for the websites:
ABUSEIPBD: https://www.abuseipdb.com OTX: https://otx.alienvault.com VirusTotal: https://www.virustotal.com

2: After signing in, copy API keys to the code.
ABUSEIPBD,VirusTotal keys in main.py OTX in is_malicious.py

3: Insert the ip after activate the code.
